Routing concepts

Standard names in the Tables below that contain instantaneous in the quantity part of the standard name can be valid for an internal time step (for example for kinematic wave routing an internal time step is optional (and recommended), with a time step equal to the model simulation time step the standard name nonetheless contains instantaneous) and thus considered instantaneous compared to the model simulation time step Δt \(\SIb{}{s}\).
